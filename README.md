# UNESCO World Heritage Sites Analysis (2019)

## 📖 Project Overview

This project presents an interactive Tableau dashboard that analyzes the UNESCO World Heritage Sites dataset (2019). The dashboard provides insights into the global distribution of heritage sites, identifies countries with the highest number of UNESCO sites, highlights endangered heritage sites, and visualizes inscription trends over time.

---

## 🎯 Project Objectives

- Analyze the distribution of UNESCO World Heritage Sites across countries.
- Identify the Top 15 countries with the highest number of heritage sites.
- Visualize the percentage of Heritage Sites at Risk.
- Explore inscription trends by year and region.
- Enable interactive analysis using country filters.

---

## 📊 Dashboard Features

- 🌍 Top 15 Countries by Number of UNESCO Heritage Sites
- 🗺️ Heritage Sites by Country (Treemap)
- ⚠️ Heritage Sites at Risk (Pie Chart)
- 📈 Heritage Sites Added by Year and Region (Line Chart)
- 🔍 Interactive Country Filter

4. Analyze the visualizations and insights.

---

## 📌 Key Insights

- China and Italy are among the countries with the highest number of UNESCO World Heritage Sites.
- Heritage sites are unevenly distributed across different countries.
- Only a small percentage of UNESCO sites are classified as "At Risk."
- The number of World Heritage inscriptions has increased steadily over time.
